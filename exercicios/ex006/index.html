Listas em HTML

Esee capítulo se trata simplemente sobre a criação de listas em HTML

Listas ordenadas

Listas ordenadas ou ordered list são todas as listas em que a ordem de seus itens é importante
EX:

  1. Acordar
  2. Orar
  3. Varrer a casa
  4. Lavar as vasilhas
  5. Estudar HTML
  6. Estudar inglês
  7. Jogar
  8. Ler a Bíblia
  9. Orar
  10. Dormir

Listas não ordenadas

LIstas não ordenadas ou unordered list são todas as listas onde a ordem de seus itens não é importante
EX:

  • Pão
  • Leite
  • Arroz
  • Feijão
  • Carne
  • Fruta

Lista de Definições

Basicamente uma lista que funciona como um dicionario

HTML
Linguagem de marcação pra criação do conteúdo de um site
CSS
Linguagem de marcação para criação do design de um site
JavaScript
Linguagem de programação para a criação de interatividade de um site
