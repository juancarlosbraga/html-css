Formatação de textos

Conjunto das aulas do capítulo 8

Principais Formatações

Negrito/Destaques

Termos podem ser colocados em Negrito ou Destaque detsa maneira utilizando a Tag <b> de maneira não semântica

Mas lembrando que em HTML5 sempre se deve dar a preferência ao sentido, ou seja devemos colocar um termo em Negrito desta maneira utilizando a Tag <strong> que é de maneira semântica

Itálico/Ênfase

Do mesmo modo com os termos em Itálico devemos priorizar a forma semântica com a Tag <em> do que a forma não semântica com a Tag <i>

Fomatações adicionais em HTML

Texto Marcado

podemos criar também um texto marcado com a tag <mark>

Texto Inserido e Deletado

Podemos utilizar a Tag <del> para indicar que um texto pode até ser lido mas desconsiderado ou utilizar a Tag <ins> para indicar que aquele trecho deve ser lido com mais atençãoo

texto Sobrescrito

Para inserir coisas como x20+3 Tag <sup>

Testo Subscrito

Para inserir coisa como H2O Tag <sub>
